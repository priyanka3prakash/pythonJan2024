## Git Process
To get the git installed version,

    git version

To get the list of branch in local, 
    
    git branch

To get the current status of the active branch

    git status

To add changes 
    
    git add <filename>

To commit the changes, 

    git commit -m "commit message"

To push the changes, 

    git push origin <branchname>
